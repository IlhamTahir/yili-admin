# YiliAdmin

YiliAdmin是一个后台解决方案，它基于Vue3与TDesign实现。它可以帮助你快速搭建企业级中后台产品原型。

## 功能特性

- [X] 登录/注销
- [ ] 权限验证
    - [ ] 动态路由
    - [X] 权限指令
- [X] 实用组件
    - [X] ECharts
    - [ ] 上传组件
        - [ ] 普通上传
        - [ ] 端点续传
        - [ ] 头像上传（裁剪上传）
- [X] 控制台
- [X] 用户管理
- [X] 角色管理
- [ ] 表格
    - [ ] 基础表格
    - [ ] 内联编辑
- [X] 错误页面
- [X] CURD模板生成器
